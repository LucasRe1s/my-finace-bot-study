# my-finance-bot — Design Spec

> **Status:** Aprovado em 25/06/2026

---

## Contexto

Bot de auxílio financeiro pessoal/familiar via **Telegram**, com entrada de dados em linguagem natural, histórico de movimentações e dashboard web. Moeda única: **BRL**.

---

## Arquitetura

```
[Telegram] ←→ [python-telegram-bot v21]
                        ↓
              [Agno Agent + Tools]
                        ↓
              [FastAPI Backend (Python 3.12)]
                        ↓
              [Supabase (PostgreSQL + Auth)]
                        ↑
              [Next.js 14 Dashboard] ←→ [FastAPI Backend]
```

---

## Componentes

| Componente | Tecnologia | Responsabilidade |
|---|---|---|
| Telegram Bot | python-telegram-bot v21 | Receber/enviar mensagens, webhooks |
| Agente IA | Agno + Claude Haiku | NLP, extração de dados, fluxos de conversa |
| API Backend | FastAPI (Python 3.12) | REST endpoints, autenticação JWT |
| Banco de dados | Supabase (PostgreSQL) | Persistência de todos os dados |
| Auth | Supabase Auth | Login email/senha + magic link |
| Dashboard | Next.js 14, Tailwind, shadcn/ui | Visualização, limites, convites |

---

## Modelo de Dados

```sql
-- Estende auth.users do Supabase
users         (id, name, telegram_id, created_at)
groups        (id, name, owner_id, created_at)
group_members (id, group_id, user_id, role: owner|member)
transactions  (id, group_id, user_id, amount, type: income|expense,
               category, description, date, created_at)
category_limits (id, group_id, category, monthly_limit, created_at)
conversations   (id, user_id, messages JSONB, updated_at)  -- últimas 10 msgs
invites         (id, group_id, email, token, accepted_at, created_at)
```

## Categorias Padrão

`Alimentação | Transporte | Moradia | Saúde | Educação | Lazer | Vestuário | Outros`

O agente mapeia texto livre para a categoria mais próxima.

---

## Fluxos de Conversa

1. **Registrar transação** → extrai valor/categoria/data/tipo → confirma → salva
2. **Consultar saldo** → "Qual meu saldo este mês?"
3. **Consultar extrato** → "Mostre meus gastos de junho"
4. **Resumo por categoria** → "Quanto gastei com alimentação?"
5. **Definir limite** → "Limite de R$ 500 para alimentação"
6. **Ajuda** → `/ajuda` lista comandos disponíveis

---

## Alertas

- Ao registrar transação: verificar se categoria atingiu 80% ou 100% do limite mensal
- Enviar mensagem Telegram ao usuário quando limite ultrapassado
